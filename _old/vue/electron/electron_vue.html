# electron-vue

# 一、 說明

electron 是開發跨平台 app 的軟體

electron-vue 則是針對 vue + webpack 使用者包裝好的一套 framework

相關說明可在下面找到

Document github

# 二、安裝及升級

因為 electron-vue 裡面的 electron 預設是 2.0 版本已經有點舊了 本文撰寫時已經是 electron 5.0.2 了 升級的時候踩了點坑，所以紀錄一下

# 1. 先照著 electron-vue 官方說明來安裝

# 安裝 vue-cli
$ yarn global add vue-cli # 或 npm install -g vue-cli
$ vue init simulatedgreg/electron-vue my-project

# 安裝 electron-vue 專案
$ cd my-project
$ yarn # or npm install #等等要升級，可以先不執行
$ yarn dev # or npm run dev #還沒要看結果，可以先不執行

# 2. 裝 yarn-check 方便升級套件 ( npm 可改用 npm-check )

$ yarn global add yarn-check # or npm install -g npm-check

# 3. 用 yarn-check 確認可升級的套件

$ yarn-check -u # or npm-check -u

這邊只選擇 electron 升級為 5.0.2 就好

# 4. 解決各種坑

升級完後就可以用 yarn dev 開啟會發現有坑

錯誤1

“Uncaught ReferenceError: require is not defined”

這個錯誤是在 chrome 的 console 出現的

查了一下 stackoverflow

發現是 electron 5.0.0 之後會預設把 nodeIntegration 關閉

所以在必須到 index.js 裡面先把 nodeIntegration 開啟


 
 
 


let win = new BrowserWindow({
  webPreferences: {
    nodeIntegration: true
  }
})

# 三、調整設定

# 1. 等載完畫面再開APP

參考

因為 electron 跟網頁一樣，先開啟 chrome 然後才開始執行前端的 javascript

這樣在render前可能會看到白色的頁面一閃而過

以下解法，就是一開始先不顯示，然後最後都載完後，再開啟APP

index.js








 


 
 
 

  mainWindow = new BrowserWindow({
    webPreferences: {
      nodeIntegration: true
    },
    height: 563,
    useContentSize: true,
    width: 1000,
    show: false
  })
  
  mainWindow.once('ready-to-show', () => {
    mainWindow.show()
  })

# 2. 設定長寬最小/大範圍

參考

APP 預設可以讓 user 調整大小，雖然通常有 RWD，

但是我們還是應該設定最小寬高，以免版面爛掉

index.js

 

 
 





 
 
 
 







  import { app, BrowserWindow, screen } from 'electron'

  const display = screen.getPrimaryDisplay();
  const dimensions = display.workAreaSize;

  mainWindow = new BrowserWindow({
    webPreferences: {
      nodeIntegration: true
    },
    maxWidth: dimensions.width,
    maxHeight: dimensions.height,
    minHeight: 300,
    minWidth: 300,
    width: parseInt(dimensions.width * 0.8),
    height: parseInt(dimensions.height * 0.8),
    icon: `${__dirname}/assets/icon.ico`,
    useContentSize: true,
    show: false
  })
