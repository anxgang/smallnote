# CUPS

使用 CUPS 建立 Print Server

# 一、 說明及安裝

Ubuntu列印和列印服務的主要機制是，通用UNIX列印系統（CUPS）。該列印系統是免費提供的便攜式列印層，已成為大多數Linux發行版中列印的新標準。

CUPS管理列印作業和佇列，並使用標準的Internet列印協定（IPP）提供遠端列印，同時支持從點陣到雷射以及介於兩者之間的各種列印機。CUPS還支持PostScript列印機描述（PPD）和遠端列印機的自動檢測，並具有簡單的基於Web的設定和管理工具。

# 使用環境如下
  1. Ubuntu 18.04

# 1. 安裝相關套件

$ sudo apt-get update
$ sudo apt-get install cups

# 二、 設定相關資料

# 1. 設定前備份

$ sudo cp /etc/cups/cupsd.conf /etc/cups/cupsd.conf.original
$ sudo chmod a-w /etc/cups/cupsd.conf.original

# 2. 設定值

$ sudo vim /etc/cups/cupsd.conf 

Listen 127.0.0.1:631 # existing loopback Listen
Listen /var/run/cups/cups.sock # existing socket Listen
# 限定本機以外其他內網主機可連線
Listen 192.168.10.250:631 # Listen on the LAN interface, Port 631 (IPP)
# 或使用 Post 不限制主機
Post 631 # Listen on port 631 on all interfaces

調整後皆需要重啟服務

$ sudo systemctl restart cups.service

# 三、 疑難雜症

# 1. 網頁顯示 403 Forbidden

表示權限不足

# Solution 1:

sudo vim /etc/cups/cupsd.conf

找到

<Location /> 
<Location /admin> 
<Location /admin/conf>

加入此行

Allow all

並重啟服務

可參考 http://ubuntuforums.org/showthread.php?t=310450

# Solution 2:

輸入此命令會改寫目前的 conf

sudo cupsctl --remote-admin

並重啟服務
